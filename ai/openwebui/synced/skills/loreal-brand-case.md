## 1.3 The commerce engine (SOP → pitch translation)
All four plans run the same machine (from ERIDU Affiliate Management SOP v1.0). This section is the shared reference for whoever builds slide B3/B8 in each deck.

**Planning cascade (§8):** GMV projection (100% base) → platform mix → content mix per platform → creator mix per platform → incentive configuration → cost model → capacity model.

**Tier system (§4):** MCN 15% / TAP 12% / Target plan 10% / Open plan 8% (defaults, configurable). Creator tiers: Mega (~10M avg monthly GMV) / Risingstar (~3M) / Mid-tier (~1M) / KOC (~50K). Umbrella rule available for organizational envelopes.
**NOTE — vocabulary match:** L'Oréal's own product tables use "%Open Plan / %Target Plan." We speak the same plan language natively. Use their terms on slides.

**Capacity model (§8.6) — the credibility weapon:** creators needed is DERIVED, never guessed:
- Realizable GMV per creator = tier avg monthly GMV × product index (default 10%, tune per brand/price point)
- Performing creators = tier GMV allocation ÷ realizable GMV
- Participating creators = performing × performance ratio (Mega 1:5 / Risingstar 1:10 / Mid 1:20 / KOC 1:50)
- Content = participating × content-per-creator (2/3/3/3 per month)
This math is what separates our GMV plan from every other bidder's hand-waved forecast. Show the formula on-slide.

**Cost model (§8.5):** commission + incentive (10% of commission) + voucher (10% of GMV) = campaign cost; reference cost ratio 22.32% of GMV; ad budget tracked separately (funding source varies). Cost ratio must be covered by vendor payout + platform incentives.

**Creator sourcing (§6):** Lists A (top 30 sellers of direct competitors) → B (brand's existing top 30) → C (category top, >1M GMV) → D (MCN inbound). Minimum 20 recruits per campaign from A–C. This is the "selection rationale" answer: we don't pick creators by follower count, we pick by demonstrated selling power against comparable products.

**Incentive triggers (§4.5):** default / EA uplift / content-target / campaign-target / monthly-target / quarterly-target → unlocks. This IS the "creator acquisition and retention strategy to build loyal selling creators" they asked for — retention through earned economics, not goodwill.

**Operating loop (§8–11):** Plan → Process (recruit, activate, govern, resolve conflicts — ack 1 day, resolve 3) → Analysis (weekly activation health, monthly performing-vs-participating actuals, quarterly tier review, per-campaign) → Optimize (reallocate incentives, tier upgrades/downgrades, vendor asks). Dashboard cadence for slide B8.

**Vendor negotiation framework (§7) — handle with care:** internally, each brand is a "vendor" and our asks escalate with our share of their affiliate performance (>5% → commission uplift ask; >10% → exclusive incentive pool; etc.). DO NOT present this as a demand ladder to the client. Reframe on-slide as a **mutual investment ladder**: "as our share of your affiliate GMV grows, here's the resourcing partnership we propose at each stage." Same content, partner framing.

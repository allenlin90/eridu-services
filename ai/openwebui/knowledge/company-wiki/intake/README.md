# Intake

Workspace for draft-ingestion before content is reviewed into `../content/`. Not synced to Open WebUI and not treated as reviewed source.

Do not commit here: raw Slack/Notion exports, personal data, credentials, or unreviewed bulk dumps. Convert to the `content/` frontmatter contract (`../tools/wiki-schema.json`) only after human review, then move the file to `../content/` and delete the intake draft.
